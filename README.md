# 🚀 AI Content Generator using n8n, Groq AI & Gmail

## 📌 Project Overview

The AI Content Generator is an automation workflow built using **n8n**, **Groq AI**, **JavaScript**, and **Gmail**. This system automatically generates high-quality content ideas and delivers them directly to the user's email without any manual effort.

The workflow uses AI to create engaging topics, attention-grabbing hooks, detailed content, and relevant hashtags for social media, blogs, marketing campaigns, and professional posts.

This project demonstrates the power of combining Artificial Intelligence with Workflow Automation to save time, improve productivity, and streamline content creation processes.

---

## 🎯 Problem Statement

Creating quality content regularly can be time-consuming and challenging. Businesses, marketers, content creators, and professionals often spend hours brainstorming topics and writing content.

This project automates the entire process by generating content ideas using AI and sending them directly to the user's inbox.

---

## ⚙️ Workflow Architecture

Schedule Trigger
↓
Groq AI (LLM Chain)
↓
JavaScript Code Node
↓
Gmail

---

## 🔄 How It Works

### Step 1: Schedule Trigger

The workflow starts automatically at a predefined time using the Schedule Trigger node.

### Step 2: Groq AI Integration

Groq AI generates:

* Content Topic
* Hook
* Detailed Content
* Relevant Hashtags

### Step 3: JavaScript Processing

A JavaScript Code node processes and formats the AI response into structured fields.

### Step 4: Gmail Delivery

The generated content is automatically sent to the user's Gmail account in a professional email format.
